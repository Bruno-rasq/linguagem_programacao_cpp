# IDEA:

Desenvolver um prototipo do genero de games "space shooter" que rode no
terminal, o game deverá contar com uma nave espacial (player) e multiplos
inimigos para combater. o jogo deve continuar enquanto o player estiver
vivo.

# OBJETIVO COM O GAME:

Praticar lógica de programação desenvolvendo um jogo "simples" usando
logicas que me vierem em mente e ajuda de inteligencia artificial.


# IMPLEMENTACOES:

- um prototipo de motor grafico rudimentar 2D.
- logica de leitura de teclas do teclado (controle).
- criação variáda de inimigos.
- controle de movimentação e ataque (player e inimigos).
- controle de memoria (pontuações recordes).


# CONTROLES:

Para capturar o estado atual da tecle pressionada optei por usar uma forma
não bloqueante possibilitando continuar o loop principal de game sem 
travar o terminal.

para isso utilizei da api windows.h os recursos de VK_Key (constantes que
representam o codigo virtual da teclas.) e o metodo GetAsyncKeyState (
retorna o bitmask representando o estado da tecla)

GetAsyncKeyState -> retora um valor de 16 bits (unsigned short) onde:
    - bit 15 (0x8000) indica se a tecla esta sendo precionada no momento.

a operação:
    GetAsyncKeyState(VK_UP) & 0x8000
    - checa se o bit mais significativo esta ativo ou não (1 ou 0).


OBS: o codigo para checagem das teclas possue uma ordem de precedencia que 
caso mais de uma tecla estejam sendo pressionada o metodo retorna o valor da primeira tecla que satisfazer a condição. (duas teclas juntas não funcionam)
